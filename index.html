Основные понятия

HTML - язык гипертекстовой разметки. Он нужен, чтобы размещать на веб-странице элементы: текст, картинки, таблицы, видео и многое другое

CSS - язык описания внешнего вида HTML-документа или же каскадная таблица стилей. В таблице стилей мы прописываем стили (правила) оформления

PHP - интерпретируемый скриптовый язык программирования общего назначения. PHP специально сконструирован для веб-разработок и его код может внедряться непосредственно в HTML

JavaScript - это язык управления элементами на сайте. Он умеет красить кнопки, запускать анимации, загружать эффекты, и всё это — без перезагрузки страницы
